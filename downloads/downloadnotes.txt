If not already done- You may need to create a credentials here:
https://www.pingidentity.com/en/account/register.html.
During Verification step under
"I'm creating an account to access..." 
choose
Product Downloads and Licensing


A) Go to https://www.pingidentity.com/en/resources/downloads/pingfederate.html
Download the PRODUCT DISTRIBUTION (ZIP)
In my case it was pingfederate-12.1.4.zip.
Copy it into downloads folder.

B) Go to https://support.pingidentity.com/s/manage-license-keys
Scroll Down. Click on "Create Free License Key" button.
Select PingFederate.
Accept the license agreement and submit
Select the license key and click View> download button.
Copy the license key into the downloads folder.
This was the file in my case- PingFederate-12.1-Development.lic

C) Go to https://www.pingidentity.com/en/resources/downloads/pingdirectory-downloads.html
Download the version of PingDirectory  from there.
In my case it was PingDirectory-10.1.0.2.zip.
Copy it into downloads folder.


D) Go to https://support.pingidentity.com/s/manage-license-keys
Scroll Down. Click on "Create Free License Key" button.
Select PingDirectory.
Accept the license agreement and submit
Select the license key and click View> download button.
Copy the license key into the downloads folder.
This was the file in my case- PingDirectory-10.1-Development.lic

E) Visit https://repo1.maven.org/maven2/mysql/mysql-connector-java/
Download the latest jar
This was the file in my case- mysql-connector-java-8.0.9-rc.jar
Copy it into downloads folder.
